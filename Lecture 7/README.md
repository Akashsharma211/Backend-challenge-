1. JEST: Jest is a JavaScript testing framework mainly used for testing:

Node.js backends

React apps

APIs

Utility functions

It was created by Meta (formerly Facebook).

command to install : "npm i jest"
for testing: "npx jest"

2. SUPERTEST: SuperTest is a library used to test HTTP APIs in Node.js.

If Jest tests functions,
SuperTest tests your API endpoints.

They are usually used together.

command to install: "npm i supertest"

3. for validating the things the package is used: npm i express-validator
